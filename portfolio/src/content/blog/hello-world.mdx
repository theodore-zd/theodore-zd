I've meant to keep a writing habit for years. This is the start.

Expect short posts on things I'm actually working on — backend performance,
system migrations, frontend architecture, and the occasional side project.

## Stack

This blog is built with [Astro](https://astro.build) content collections + MDX,
styled with Tailwind v4, and deployed as a static site. Posts live in
`src/content/blog/` as `.mdx` files. Zero CMS, fast to ship, easy to grep.
